Privacy Policy

Last updated: June 12, 2026

1. Overview

EstateSync is an integrated Housing Society and Plot Management System designed to manage residents, plots, payments, and administrative operations. We are committed to protecting the privacy and security of all users including residents, agents, and society administrators.

2. Information We Collect

To provide smooth system functionality, EstateSync collects the following data:

  • Full Name and Contact Information
  • Email Address for account authentication and notifications
  • User Role (Resident, Agent, or Administrator)
  • Society and plot-related records (if applicable)
  • Transaction and payment history within the system
3. Purpose of Data Usage

The collected information is used strictly for system operations including:

  • Managing user authentication and account access
  • Maintaining society records and plot allocations
  • Processing and tracking payments securely
  • Generating reports for administrators
  • Improving system performance and user experience
4. Data Protection & Security

EstateSync implements secure database management practices to protect user data from unauthorized access, alteration, or loss. Sensitive information such as passwords is stored in encrypted form within the system.

5. Role-Based Access Control

Access to data within EstateSync is restricted based on user roles:

  • Residents: Can only view their own profile, payments, and plot details
  • Agents: Can manage assigned plots and related records
  • Administrators: Have full system access including user and financial management
6. Data Sharing Policy

EstateSync does not sell, rent, or share user data with third-party organizations. Data is only shared internally within the system for operational requirements.

7. User Rights
  • View and update personal profile information
  • Request correction of incorrect data
  • Request account deactivation (admin approval required)
8. System Logs

For security and monitoring purposes, EstateSync maintains system logs including login attempts, transactions, and administrative actions. These logs are used only for auditing and system protection.

9. Contact Information

For any privacy-related concerns, users may contact:

Email: support@estatesync.com

System: EstateSync Administration Panel
